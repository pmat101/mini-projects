Section 2: An Introduction to Web Development

  • 011.    The 'internet' is the infrastucture that connects billions of different devices all across the globe using a method called routing where individual machines are identified by their IP address, it is what makes the working of web, email, streaming, gaming possible.
  • 012.    World Wide Web is a system that allows exchange of documents over the internet using the HTTP protocol, according to which a client sends a request and receives a response from a server identified by its URL.
  • 013.    The response received from a server is in the form of HTML-CSS-JS files which are interpreted by the client's browser and then displayed.
  • 014.    Server-side logic = backend, Client-side code = frontend
  • 015.    HTML = noun, CSS = adjective, JS = verb
